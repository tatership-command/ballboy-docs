# Command Reference

Ball Boy registers twelve top-level slash commands. Each will get its own reference
page covering its subcommands, options, permission gate, and effect:

- `/league` — create, configure, inspect, and delete a league; connect it to the EA
  companion app.
- `/season` — create and manage seasons: schedule import, results, advancing,
  rollback.
- `/team` — claim, switch, release, or inspect team ownership.
- `/teams` — show the persistent roster board.
- `/standings` — show current standings in Discord.
- `/schedule` — show the game schedule in Discord.
- `/game` — manage an individual game thread: scheduling and thread deletion.
- `/admin` — server-owner configuration: channels, roles, auto-advance, welcome,
  board appearance, and permission preflight.
- `/waitlist` — manage the team-claim waitlist.
- `/welcome` — post the "how to claim a team" help message.
- `/stream` — announce a stream to your league.
- `/draft` — run a team draft: setup, join, pick, and the on-clock timer.

Reference pages for each command are being authored.
