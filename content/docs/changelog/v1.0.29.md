# v1.0.29 — Team Roles Recolor on Sync

_Released 2026-07-11_

## Fixes

- **`/admin roles sync_all` now repaints existing team and conference role colors.** Previously a role's color was only set when the role was first created, so changing a team's color (or fixing a hard-to-read one) never updated roles that already existed. Sync now recolors any role whose color is out of date. After a color change, run `/admin reload_teams` then `/admin roles sync_all` and the roles update in place.
