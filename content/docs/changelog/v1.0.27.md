# v1.0.27 — Draft Order Race

_Released 2026-07-11_

## What's New

- **Race for your draft order — live, in the Activity.** When setting up a draft, pick "🎲 Race for order" instead of manual order. When you're ready, open the Ball Boy Activity and press Start Race to run a live football-lane race that decides the pick order in front of everyone.
- **Watch it happen, then lock it in.** Nothing is decided until you're ready — press Race Again as many times as you like, then press Set Draft Order to make it official. That's the moment the pick order is saved and the draft board posts.
- **See real faces, not numbers.** Race lanes now show each manager's Discord picture and name instead of a raw ID.
- **Watch the order fill in live.** As each racer crosses the finish line, their pick spot appears immediately — no more waiting for the whole race to finish before you see who's picking first.

## Fixes

- The draft board no longer shows a frozen countdown clock before the draft has actually started.
- Ownership changes from an auto-picked team now show up on `/teams` right away.
- Fixed a rendering glitch where an empty warning box could appear on the Draft Race screen.
