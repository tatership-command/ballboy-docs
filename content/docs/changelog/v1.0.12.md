# v1.0.12 — Owner superuser + cleaner dynasty roles

_Released 2026-07-05_

## What's New
- **Bot owner can run any command** in any server — no more permission bounces.
- **Conference roles are set up cleanly** — created all at once when a season is created, named in full (e.g. `Big Ten`, not `B1G`), and existing conference/team roles are reused instead of duplicated.

## Fixes
- **`/admin roles sync_all` now removes stale roles** — members who no longer own a team lose that team's role and its conference role (including for teams that are now CPU).
