# v1.0.38 — League Staff Access — Grant Commissioners & Admins from Discord

_Released 2026-07-17_

## What's New

**🛡️ New `/admin access` command**
Commissioners can now grant or revoke **commissioner** and **admin** access for any member straight from Discord. It updates their access and assigns (or removes) the matching server role automatically.

**🔓 Staff roles now unlock the in-app tools**
If you hold your league's commissioner or admin role, the management tools in the Ball Boy Activity (Manage Schedule, Custom Teams) now appear for you — previously only the league's creator could see them. Run **`/admin roles sync_all`** once and Ball Boy will bring all your existing staff up to date automatically.
