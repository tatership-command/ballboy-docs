# v1.0.22 — /stream Access Fix

_Released 2026-07-08_

## Fixes

- **`/stream` no longer wrongly denies league members.** Ordinary team owners were getting "You do not have permission to use this command" because the command required an access level most members don't hold. It now correctly allows any member of the league's Discord server to announce a stream.
