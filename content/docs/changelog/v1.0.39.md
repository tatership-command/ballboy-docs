# v1.0.39 — Channel Permission Checks & a Guided Setup Nudge

_Released 2026-07-17_

## What's New

**⚠️ `/admin channels` now checks its work**
When you point Ball Boy at a channel it can't post in, it still saves your choice — but now it tells you right away which channel and exactly which permission is missing, so nothing silently fails later. It even knows the difference between a text channel and a forum `game_thread`, so a correctly set-up forum won't get flagged.

**🧭 A nudge toward the next step**
Ball Boy's setup permission message — the one you see when it joins your server and whenever you run **`/admin roles_setup`** — now reminds you to set your admin & commissioner roles, grant access, and route your channels, with a link to the full **Your First League** setup guide.
