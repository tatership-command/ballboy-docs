# v1.0.42 — Schedule Builder No Longer Deletes Games You Didn't Touch

_Released 2026-07-24_

## Fixes

**🗓️ Editing one team's schedule no longer removes another team's game**
In the Manage Schedule Activity, building or editing a schedule could accidentally delete a *different*, unrelated team's already-scheduled game — and its Discord thread along with it. For example, adding a new team's slate could quietly remove a matchup that was already set up for someone else. Applying a schedule now changes **only the games you actually edited** — every matchup you didn't touch is left exactly as it was.
