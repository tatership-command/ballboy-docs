# v1.0.36 — CFP game banners

_Released 2026-07-16_

## What's New
- **CFP game threads now open with a banner.** Playoff game threads — First Round, the New Year's Six bowls, and the National Championship — get a broadcast-style graphic with the round/bowl artwork, both team logos, seeds, and records, created automatically. The National Championship banner fills in the season's year on its own.
- **Smarter custom-team lookup.** Adding a league team now merges College Football Data and ESPN into a single result per team, flags any missing info, and lets you overwrite a custom team you already added. ESPN men's basketball is available as a lookup source too.
- **Faster Activity sign-in.** Returning users skip the per-launch sign-in button — Ball Boy re-authenticates silently.

## Fixes
- **First-round labels.** First-round playoff threads now read "CFP First Round" instead of the host school's campus-site venue name.
- **Quieter auto-advance.** With auto-advance enabled, the weekly advance announcement no longer pings again (@everyone or the announce role) — the advance trigger already notified everyone.
