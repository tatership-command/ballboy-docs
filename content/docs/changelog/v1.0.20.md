# v1.0.20 — /stream Upgrades — Pick a Profile, Stream in Discord

_Released 2026-07-07_

## What's New

**`/stream` now works the way you'd expect** — more control over what gets announced.

- **Pick which profile to link** — a new `profile` option lets you choose **Twitch**, **YouTube**, or your **Custom URL** when you have more than one saved. Leave it off to show all of them.
- **Streaming in Discord?** — a new `voice_channel` option announces a 📺 link straight to the voice channel you're screen-sharing in, no external stream needed.
- **Multiple links now stack on their own lines** instead of running together, so they're easy to click.

## Fixes

- **Custom stream URLs from the Claim Activity now actually show up.** Previously a custom URL you saved was stored but never surfaced in `/stream` or Go Live — only your Twitch/YouTube links appeared. It now saves to your profile and works everywhere. If you set a custom URL before this update, re-open the Claim Activity once and re-enter it to sync it.
