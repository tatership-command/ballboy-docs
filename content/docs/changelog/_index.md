# Release Notes

Every shipped version of Ball Boy gets a release note here, newest first. These are
the same notes Ball Boy posts to `#announcements` when a version deploys.

Each entry is color-coded in Discord (not shown on this page): **green** releases
add new features, **orange** releases are fixes or maintenance. Entries are added
here automatically as part of the release process — you don't need to look
anywhere else for "what changed and when."
