# v1.0.23 — Stream Link Fix

_Released 2026-07-08_

## Fixes

- **`/stream` and Go Live now build working Twitch/YouTube links.** Links pulled from your saved profile were doubling the domain (e.g. `https://twitch.tv/twitch.tv/yourname`), so the URL didn't work. They now resolve to the correct address. No need to re-link — already-saved profiles are fixed automatically.
