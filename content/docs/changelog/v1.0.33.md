# v1.0.33 — Draft reliability — pick without timeouts, and drafters auto-join

_Released 2026-07-13_

## What's New
- **Drafters can pick right away.** Adding someone to a draft — in `/draft setup`, or when they run `/draft join` — now makes them a league member automatically, so their team list shows up immediately in `/draft pick`. Leaving or being removed from the draft removes that membership again (unless they already own a team).

## Fixes
- **No more "Unknown interaction" timeouts.** Commands that do real work — especially `/draft pick` during a busy draft — no longer time out and show a false error. Every command now acknowledges instantly, so your pick registers and the draft board stays in sync.
