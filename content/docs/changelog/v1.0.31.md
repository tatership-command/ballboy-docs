# v1.0.31 — Notification controls, role tools, and fixes

_Released 2026-07-13_

## What's New
- **Game notifications, your way.** Two opt-in buttons on the weekly announcement let members follow **All games** or **only User-vs-user games**. Commissioners choose how those notifications arrive with the new **`/admin notify_mode`** — a quiet role ping (default) or adding followers to each game thread.
- **Create roles right from `/admin roles`.** Type a role name and Ball Boy will create it — or reuse a matching one — after a quick confirm. No more setting up roles by hand first. New roles use your league color and stay out of the way (not shown separately, not mentionable by everyone).
- **Cleaner roles in multi-league servers.** Team and conference roles are now scoped per league, so running more than one league in a server no longer crosses wires.
- **Activity buttons show they're working.** Apply and Confirm buttons now show a spinner instead of looking frozen while they save.

## Fixes
- New-member welcome messages now fire only when someone actually gains access — no more welcome spam after a role sync.
- Rolling back a week now cleans up that week's game threads.
- A deleted game thread no longer blocks the season from advancing.
