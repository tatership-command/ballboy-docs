# v1.0.28 — Readable Team Colors & Clearer Schedule Cards

_Released 2026-07-11_

## What's New

- **Team colors that actually show up on dark mode.** 30 teams whose primary color is a dark navy (Michigan, Penn State, Auburn, Notre Dame, Navy, Pittsburgh, West Virginia, and more) now use their brighter secondary color for roles and embeds, so they no longer disappear against Discord's dark theme. Run `/admin reload_teams` then `/admin roles sync_all` to repaint existing team roles.
- **Schedule cards show who owns each team.** In the Manage Schedule builder, each team card now shows the owner's Discord name instead of a raw ID number.

## Fixes

- Documentation site sync now runs on release, keeping the public docs current with each version.
